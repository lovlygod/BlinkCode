## Основы редактора

| Сочетание | Действие |
|---|---|
| `Ctrl+S` | Сохранить текущий файл |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Повторить |
| `Ctrl+X` | Вырезать строку (без выделения) / выделение |
| `Ctrl+C` | Скопировать строку (без выделения) / выделение |
| `Ctrl+V` | Вставить |
| `Ctrl+A` | Выделить всё |
| `Tab` / `Shift+Tab` | Отступ / убрать отступ |
| `Ctrl+/` | Закомментировать строку |
| `Shift+Alt+A` | Закомментировать блок |
| `Alt+↑` / `Alt+↓` | Переместить строку вверх / вниз |
| `Shift+Alt+↑` / `Shift+Alt+↓` | Скопировать строку вверх / вниз |
| `Ctrl+Enter` | Вставить строку снизу |
| `Ctrl+Shift+Enter` | Вставить строку сверху |
| `Ctrl+Shift+K` | Удалить строку |

## Навигация

| Сочетание | Действие |
|---|---|
| `Ctrl+P` | **Quick Open** — нечёткий поиск файлов |
| `Ctrl+Shift+P` | **Command Palette** |
| `Ctrl+G` | Перейти к строке |
| `Ctrl+Shift+O` | Перейти к символу в файле (outline) |
| `F12` | Перейти к определению |
| `Ctrl+Click` | Перейти к определению |
| `Alt+F12` | Peek-определение |
| `Shift+F12` | Найти все использования |
| `Ctrl+Home` / `Ctrl+End` | Начало / конец файла |
| `Ctrl+←` / `Ctrl+→` | По словам |
| `Ctrl+↑` / `Ctrl+↓` | Прокрутка без перемещения курсора |

## Выделение и мульти-курсор

| Сочетание | Действие |
|---|---|
| `Alt+Click` | Добавить курсор в точке клика |
| `Ctrl+Alt+↑` / `Ctrl+Alt+↓` | Добавить курсор выше / ниже |
| `Ctrl+D` | Добавить следующее вхождение в выделение |
| `Ctrl+Shift+L` | Выделить все вхождения |
| `Shift+Alt+drag` | Column-выделение (прямоугольник) |
| `Ctrl+L` | Выделить текущую строку |
| `Ctrl+Shift+\` | Перейти к парной скобке |

## Поиск и замена (в файле)

| Сочетание | Действие |
|---|---|
| `Ctrl+F` | Найти |
| `Ctrl+H` | Заменить |
| `F3` / `Shift+F3` | Следующее / предыдущее совпадение |
| `Alt+Enter` | Выделить все совпадения |
| `Esc` | Закрыть виджет поиска |

## Языковая поддержка (LSP)

См. [lsp.md](./lsp.md) — что делает каждая команда «под капотом».

| Сочетание | Действие |
|---|---|
| `Ctrl+Space` | Вызов подсказок (автокомплит) |
| `Ctrl+Shift+Space` | Подсказки параметров (signature help) |
| Hover | Показать типы и документацию |
| `F12` | Перейти к определению |
| `Shift+F12` | Найти все использования |
| `Ctrl+Shift+O` | Символы документа / outline |
| `Ctrl+.` | Quick fix / code actions (add missing import, organize imports и т.д.) |

## Рефакторинг

| Сочетание | Действие |
|---|---|
| `F2` | Переименовать символ (во всех файлах) |
| `Ctrl+.` | Меню code actions |

## Форматирование

| Сочетание | Действие |
|---|---|
| `Shift+Alt+F` | Форматировать документ |
| `Ctrl+K Ctrl+F` | Форматировать выделение |

## Терминал и панели

| Сочетание | Действие |
|---|---|
| Клик по иконке в activity bar | Переключение панели (файлы / поиск / AI / настройки) |
| `` Ctrl+` `` *(если привязано)* | Показать / скрыть терминал — [`Terminal`](../../src/components/Terminal/Terminal.tsx) |
| Клик по URL в терминале | Открыть во встроенном [`BrowserPreview`](../../src/components/BrowserPreview/BrowserPreview.tsx) |

## Окно и приложение

| Сочетание | Действие |
|---|---|
| `Alt+F4` | Закрыть окно |
| Window controls | Кастомный titlebar — [`TopHeader`](../../src/components/TopHeader/TopHeader.tsx) |

## DevTools

| Сочетание | Действие |
|---|---|
| `F12` (вне области редактора) | Переключить DevTools *(также `Ctrl+Shift+I`)* |
| `Ctrl+Shift+I` | Переключить DevTools |
| Авто-открытие в dev | DevTools открываются автоматически при `npm run electron:dev` |

> ⚠️ `F12` используется **и** Monaco (перейти к определению), **и** Electron
> DevTools. Внутри области редактора работает первое, вне редактора (сайдбар,
> терминал, панели) — второе.
