# Contributors

## Project

[`BlinkCode`](README.md) is developed as an open-source desktop-first code editor project.

## Current Contributors

### Core

- [`lovlygod`](https://github.com/lovlygod) — creator and primary maintainer

## How to Contribute

If you want to contribute to [`BlinkCode`](README.md), you can help with:

- bug fixes
- UI and UX improvements
- desktop packaging improvements
- terminal stability
- file handling and editor features
- documentation
- translations

## Contribution Flow

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test the changes locally
5. Open a pull request

## Notes

- Keep commits focused and clear
- Prefer changes that match the existing code style
- Document user-facing changes when needed
